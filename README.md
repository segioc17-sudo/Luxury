Luxury World — Vite + React

Este repo contiene la tienda Luxury World creada con Vite + React.

Build

```bash
npm install
npm run build
```

Deploy en Vercel

Opción 1 — Importar desde Git: subir el repo a GitHub y luego *Import Project* en Vercel.

Opción 2 — Vercel CLI:

```bash
npm i -g vercel
vercel login
vercel --prod
```

Ajustes de build: `Build Command`: `npm run build`, `Output Directory`: `dist`.
